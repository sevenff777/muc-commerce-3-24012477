## 用自己的话回答

- 什么是特征，什么是标签：特征是用来做判断的原始数据，比如用户使用时长、消费金额、是否投诉、登录设备类型这类用户属性，是模型推理的依据，记作 X。标签是我们最终想要预测的目标结果，本项目里就是用户是否流失（Churn：0 不流失 / 1 流失），也就是模型要学习回答的标准答案，记作 y。
- 为什么要保留测试集：训练集只用来给模型学习数据规律，如果只用训练集评估效果，模型会记住样本造成过拟合。测试集全程不参与模型训练，模拟线上全新未知用户，用来真实检验模型对陌生数据的泛化能力，判断它能不能在实际场景里正常预测。
- 为什么83%准确率仍可能没有用：本次数据集整体流失率仅 16.84%，绝大多数用户都是不流失。无脑全部预测 “不流失” 就能拿到约 83% 的准确率，但完全识别不出真正要流失的用户（召回率为 0），无法实现挽留流失客户的业务目标。高准确率只是样本分布倾斜带来的假象，模型没有实际业务价值。
